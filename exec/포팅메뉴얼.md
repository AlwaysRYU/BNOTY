### 빌드 세팅 상세

------

- FireBase Setting

  - ```js
    const firebaseConfig = {
      apiKey: "",
      authDomain: "",
      databaseURL:
        "",
      projectId: "",
      storageBucket: "",
      messagingSenderId: "",
      appId: "",
      measurementId: "",
    };
    ```

  - Firebase.js 첫번째 줄 firebaseConfig 정보 입력
  - 해당 내용은 **파이어베이스 콘솔 → 프로젝트 설정 → 일반** 에서 확인가능 



- 확장 프로그램 빌드
  - [chrome://extensions/](chrome://extensions/ ) 로 접속
  - 우측 상단의 개발자 모드 **ON**
  - ![image-1](https://i.ibb.co/YBwR0c5/ddd.png)
  - **압축해제된 확장 프로그램을 로드합니다** 클릭
  - **BNOTY 폴더** 선택 후 사용
